An Intro I wrote about My First Issues with Ghost Blog in 2013

Well, I just wrote over 2 pages of text and it was all wiped out by an accidental slip of the mouse button. Guess there is no auto-save feature? I will have to check that out.

Lets see if I can shorten this:

Ghost is a new blogging platform and I didn't really have the time (I technically still don't) to try and install it. It has taken a solid 1.5 hours to get it up and running on my little server but it seems to be working.

There are 2 possible issues. The first is I am not sure if I am saving this in mysql or sqlite. It is suppose to be mysql. The second is the upstart functionality. When I reboot my server I can't seem to verify ghost will restart. Again, this is something I will have to test later.

Lastly, this is being housed on an atom server running on Ubuntu 13.10.

I repurposed an Acer Veriton with an atom processor. Replaced the expensive fan and added an SSD and doubled the memory. It also serves as a remote desktop host with virtualbox running an XP guest. I also occasionally use it to stream video to the TV. It sits behind the TV and is silent. You may think that putting an SSD in it is overkill but when you are paying 52.5 cents (soon to go up another 2.5 cents) per Kwh you would also be doing everything to reduce your electricity costs.
