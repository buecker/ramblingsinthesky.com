WooCommerce site with Astra Died

While I've never been a fan of WordPress, it does offer an affordable way to host an e-commerce site. Unfortunately, my e-commerce site recently encountered significant issues. It was utilizing WooCommerce for the e-commerce functionality and Astra for the theme. Lately, I've been experiencing difficulties with Elementor bundled with Astra. Just yesterday, the site became completely inaccessible. After some troubleshooting, I managed to get it running briefly before it crashed again. Even the recovery mode isn't functioning properly at this point.

This is a sign that I need to switch so something else. I plan to see if I can get medusajs to work. I tried for a short while yesterday but failed miserably.
